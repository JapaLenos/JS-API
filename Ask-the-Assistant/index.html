À propos de l'application

Cette application tire parti des composants d'IA du SDK JavaScript d'ArcGIS pour proposer une expérience interactive d'exploration des données d'accidents corporels de la circulation routière.

Les données ont été produites par le Département des Hauts-de-Seine.

Retrouvez toutes les étapes de création de l'application dans cet article CodeTheMap, et toutes mes autres apps sur Github @JapaLenos.
